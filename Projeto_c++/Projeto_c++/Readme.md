# Mini Sistema RH — C++ (Férias / Faltas)

-Projeto simples em C++ para gerir colaboradores, marcar férias/faltas e visualizar um calendário mensal.

- Os colaboradores são guardados num ficheiro TXT e os nomes ficam encriptados com uma cifra de César (shift +3).

- Funcionalidades

- Adicionar colaboradores

- Listar colaboradores

- Marcar férias (F)

- Marcar faltas (X)

- Remover marcações

- Mostrar calendário mensal com indicadores

- Legendas do calendário:

- F = Férias

 - X = Falta

- (dia) = Fim de semana

- Como correr localmente

- Sim, pode correr localmente.

- Necessário apenas um compilador C++ como por exemplo:

- Visual Studio (usei Visual Studio 2026 Insiders)
